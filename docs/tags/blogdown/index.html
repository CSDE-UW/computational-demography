Making a Research Website

   2019-02-14      tutorial      blogdown · GitHub

Why make a website? Academic researchers need to communicate their work! Several kinds of websites can be useful for this: a personal academic homepage a page for a specific research project a website for a course a website for a working group or seminar (like CDWG!) Key ideas You can make a website without knowing much or any html/css; there are tools that let you do a lot in plain text and R instead.
